Design a responsive single page site.

You will design a mobile version (320px), tablet version (768px), small desktop version (1024px), and large desktop version (1440px).

The content of the site is up to you but you should have text, images, and links to external sites.

The digital versions should be built in Illustrator, XD, or Sketch.

Some ideas could be a site for a current issue, a site re-presenting a longform article, a site acting as a type specimen, or any some other content source.
